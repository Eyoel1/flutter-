## Project Overview

This repository contains the implementation of the first programming assignment for the Dart module. The project is divided into two main tasks that demonstrate core Dart programming concepts.

### Task 1: Number Analysis App

A Dart program that performs manual analysis on a list of integers. It includes functions to find the maximum, minimum, sum, and average of a list without using built-in high-level collection methods, demonstrating a mastery of basic loops and control flow.

- **Location:** `task1/number_analysis.dart`
- **Bonus Features:** Null-safe empty list handling, Count Negatives function, and Bubble Sort implementation.

### Task 2: Async Calculator App

An object-oriented Dart application that simulates an asynchronous calculator. It features a `Calculator` class with arithmetic operations, exception handling for division by zero, and asynchronous processing using `Future`, `async`, and `await` to simulate network delays.

- **Location:** `task2/calculator_app.dart`
- **Bonus Features:** Calculation History Log, Unknown Operation exceptions, Parallel Execution using `Future.wait()`, and Chained Operations.
